Phase 1 — Créer le dépôt GitHub
1

Crée un compte GitHub si pas encore fait

Va sur github.com → Sign up → entre ton email → crée un mot de passe → vérifie ton email.

ℹ️ Si tu as déjà un compte GitHub, passe directement à l'étape 2.
2

Crée un nouveau dépôt (repository)

  • Clique le + en haut à droite → "New repository"
  • Nom du dépôt : akili-digital
  • Visibilité : Public (obligatoire pour GitHub Pages gratuit)
  • Coche "Add a README file"
  • Clique "Create repository"
3

Upload les 4 fichiers HTML

Dans ton dépôt → clique "Add file" → "Upload files"

  • Glisse ou sélectionne ces 4 fichiers :
# Les 4 fichiers à uploader ensemble

index.html ← entrée principale
akili-digital-sales-page.html ← page de vente
acces-X7k3rjWAbxkTUL-6.html ← livraison secrète
business-ai-planner-2026.html ← le produit
  • Message de commit : "Tunnel de vente Akili Digital"
  • Clique "Commit changes"
Phase 2 — Activer GitHub Pages
4

Active GitHub Pages

  • Dans ton dépôt → onglet "Settings"
  • Menu gauche → "Pages"
  • Source : "Deploy from a branch"
  • Branch : main → dossier : / (root)
  • Clique "Save"
⏳ Attends 2-3 minutes. GitHub Pages génère ton site automatiquement.
5

Récupère ton URL définitive

Retourne dans Settings → Pages. Tu verras :

github.com/TON_USERNAME/akili-digital/settings/pages

https://TON_USERNAME.github.io/akili-digital/
